# Midlands AutoCare

A Next.js portfolio demo for NorthSummit Agency, built as a full redesign of a fictional Nottingham MOT centre and independent garage.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Vitest + Testing Library

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run test
npm run build
```
