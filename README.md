# Midlands AutoCare (Garage Landing Page)

A conversion-focused, mobile-first garage website built with Vite + React + TypeScript.

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run test
npm run build
```

## Cloudflare Pages

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Node: 20
